第一步：使用font-face声明字体 
@font-face {font-family: '5ufont';
    src: url('5ufont.eot'); /* IE9*/
    src: url('5ufont.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
    url('5ufont.woff') format('woff'), /* chrome、firefox */
    url('5ufont.ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
    url('5ufont.svg#5ufont') format('svg'); /* iOS 4.1- */
}
第二步：定义使用iconfont的样式 
.iconfont{
    font-family:"5ufont" !important;
    font-size:16px;font-style:normal;
    -webkit-font-smoothing: antialiased;
    -webkit-text-stroke-width: 0.2px;
    -moz-osx-font-smoothing: grayscale;}
第三步：挑选相应图标并获取字体编码，应用于页面 
<i class="iconfont">&#x33;</i>
// 或者
<i class="iconfont icon-check-square"></i>
